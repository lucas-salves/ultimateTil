# Git remove a remote

Syntax:
```
git remote remove [name of the remote]
```
Example:
```
$ git remote -v
origin	https://github.com/lucas-salves/ultimateTil.git (fetch)
origin	https://github.com/lucas-salves/ultimateTil.git (push)

$git remote remove origin
```
