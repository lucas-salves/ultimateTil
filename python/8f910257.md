# Nested dictionaries

Nested dictionaries are dictionaries inside dictionaries.

Example below create a dictionary that contain three dictionaries:
```
myfamily = {
	"child1": {
		"name":"Emil",
		"year":2004
	},
	"child2":{
		"name": "Tobias",
		"year": 2007
	},
	"child3":{
		"name":"Linus",
		"year": 2011
	}
}
```
If you want to add three dictonaries into a new dictionary:
```
 child1 = {
  "name" : "Emil",
  "year" : 2004
}
child2 = {
  "name" : "Tobias",
  "year" : 2007
}
child3 = {
  "name" : "Linus",
  "year" : 2011
}

myfamily = {
	"child": child1,
	"child2": child2,
	"child3": child3
}
```
