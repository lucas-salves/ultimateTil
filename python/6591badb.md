# Copy dictionaries

## copy() method
make a copy of a dictionary:
```
thisdict =	{
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = thisdict.copy()
```

## dict() function
You can use too dict() function
```
mydict = dict(thisdict)
```
