# Assign multiple values

Assign the same values to more than one variable
```
x = y = z = "Orange", "Banana", "Cherry"
print(x)
print(y)
print(z)
```
Python allows to extract list, tuple, etc values into variables this way:
```
fruits = ["apple","banana","cherry"]
x,y,z = fruits
print(x)
print(y)
```
