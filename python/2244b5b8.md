# Escape

Is used to insert characters that are illegan in String

Example:
Adds a double quote inside a string surrounded by double quotes:
```
txt = "We are the so-called \"Vikings\" from the north."
```

Other escape characters:

| Code | Result  |
| :---:   | :-: |
|\\'       | single quote |
|\\\       |backslash  |
|\\n       |new line  |
|\\r       | carriage return |
|\\t       |tab  |
|\\b       | backspace |
|\\f       | form feed |
|\\ooo     |octal value  |
|\\xhh     |hex value  |
