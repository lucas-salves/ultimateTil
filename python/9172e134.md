# Modify Strings

Some of the built-in methods to modify strings

# Upper case
```
a = "Hello, world!"
print(a.upper())
```
# Lower case
```
a = "Hello, world!"
print(a.lower())
```
# Remove whitespace
```
a = "Hello, world!"
print(a.strip())
```
# Replace string
```
a = "Hello, world!"
print(a.replace("H","J"))
```
# Split string
```
a = "Hello, world!"
print(a.split(",")) # returns ['Hello', 'World!']
```
