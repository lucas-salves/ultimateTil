# Join tuples

## Join 2 tuples using the + operator:

```
tuple1 = (a"","b","c")
tuple2 = (1,2,3)

tuple3 = tuple1+tuple2
print(tuple3)
```
## Multiply tuples
Is possible to multiply the content of a tuple using the * oeprator:
```
fruits = ("apple", "banana", "cherry")
mytuple = fruits * 2

print(mytuple)
```
