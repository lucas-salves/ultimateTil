# Throw a exception without breaking a loop

Single exception

```
Exception ex = null;

for(;;){
	ex = new Exception();
}

if(ex != null)
	throw ex;
```
List of exceptions

```
List<Exception> errors = new ArrayList<>();
for(;;){
	errors.add(new Exception());
}

if(!errors.isEmpty()){
	// Do something with the errors
}
```
