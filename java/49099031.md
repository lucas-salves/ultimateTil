# Truncating BidDecimal objects

```
//1
new BigDecimal("35.3456").round(new MathContext(4, RoundingMode.HALFUP));
//result = 35.35


//2
new BigDecimal("35.3456").setScale(2, RoundingMode.HALF_DOWN);
// result = 35.35
```
