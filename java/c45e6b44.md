# Interpolate Strings

Use *printf*:

```
String nome = "Fulano";
int idade = 20;

System.out.printf("%s tem %d anos \n",nome, idade); //Fulano tem 20 anos
```
This syntax is inpired by C. The *%s* and *%d* represents respectively:
- A String and a integer number.
